## 確率的ボラティリティ(SV)モデル

**データ**: 日経225 対数収益率(2015-2026)

| バージョン | 概要 | 結果 |
|---|---|---|
| Baseline | non-centered AR(1)-SV、正規分布観測 | $\phi\approx0.91$, $\sigma_\eta\approx0.36$。ボラティリティクラスタリングを概ね再現するが、$r_t^2$のACF lag=1で過小評価 |
| Student-t観測 | 裾の重い分布に変更 | $\nu\approx14$、lag=1のギャップは未解消(観測分布の問題ではなかった) |
| Leverage効果 | $\epsilon_t,\eta_t$に相関$\rho$を導入 | $\rho\approx-0.54$と有意だが、lag=1のギャップは依然未解消 |
| 2-factor(fast/slow) | 2つの時間スケールの重ね合わせ | lag=1のギャップがほぼ解消。ただしfast/slow間に緩やかな非識別性(σのridge構造)が残存 |

**設計上の要点**: non-centered parameterization(`pm.AR(sigma=1.0)` + 外側でスケール)によりGaussianRandomWalkの非平均回帰性を回避。

## マルコフ転換モデル(MSM)

**データ**: 日経225(離散2レジーム、平時/危機)

- Forward algorithmで離散潜在状態を周辺化し、`pm.Potential`で対数尤度を直接記述(離散変数をNUTSで直接サンプリングしない定石)。
- 弱情報事前分布に切り替えた際、2レジームのボラティリティがほぼ同値に収束する**ラベルスイッチング**が発生。divergence=0・r_hat≈1.00でも検出できない典型例。
- `pt.sort()`による順序制約でラベルスイッチングを解消。最終的に、ボラティリティの水準差は明確にデータから支持されるが、遷移確率(レジームの持続性)は弱情報事前分布下ではデータからほとんど特定できないと判明。

## 階層ベイズモデル

3つの共役事前分布ファミリーで、"平均・集中度"への再パラメータ化とshrinkageの効き方を比較。

| 分布族 | データ | κ(集中度): 事前→事後 | shrinkageの強さ |
|---|---|---|---|
| Beta-Binomial | MLB打率(174選手) | 100→600 | 強い |
| Gamma-Poisson | サメ襲撃件数(13ヶ国) | 6→0.81 | 弱い |
| Dirichlet-Multinomial | ポケモンカード封入率(8商品) | 100→229 | 非常に強い |

**共通する設計原則**: $\alpha=\mu\kappa$(Beta)、$\beta=\alpha_{conc}/\mu$(Gamma)、$\alpha=\kappa\pi$(Dirichlet)という形で、分布ごとに実装は異なるが「方向・集中度」の分解思想は一貫。shrinkageの強さはデータの均質性をそのまま反映することを3ケースで確認。

## 点過程モデル

**データ**: 能登半島地震(2024)、Hawkes過程(ETAS型自己励起点過程)

$$\lambda(t)=\mu+\sum_{t_j<t}\kappa e^{-\beta(t-t_j)}$$

- 尤度を`pm.Potential`で直接記述(`log L = Σlog λ(t_i) - ∫λ(t)dt`)。時間差行列のベクトル化(`dt[:,None]-dt[None,:]`+マスク)を実装。
- $\kappa,\beta$間に強いridge構造(非識別性)がありdivergence(53件)発生。分岐比$M=\kappa/\beta$への再パラメータ化で緩和(divergence 53→36)。
- $M\approx0.74$(89%区間[0.62,0.87])と1未満を明確に推定、余震活動が安定的(subcritical)であることを定量的に確認。

## 感染症コンパートメントモデル

SIR→SIS→SIRS→SEIRの順に難易度を上げながら4モデルを実装。

### SIR (Eyamペスト、1666年)

- 8時点の疎データから$\beta\approx5.04$, $\gamma\approx3.17$、$R_0\approx1.59$を推定。原論文(Raggett 1982, $R_0\approx1.9$)と近いオーダーで独立に再現。
- ピーク時点は完全一致、ピーク高さはやや過小評価(Poisson観測分布の「分散=平均」制約による可能性)。

### SIS (ナイジェリア・マラリア罹患率、2015-2024)

- 地方病的平衡(endemic equilibrium)への到達を表現。平衡水準の解析式$\text{incidence}^*=\gamma N(1-1/R_0)$から必要な$R_0\approx1.05$を逆算し、$R_0=1+\epsilon$という再パラメータ化で暴走を解消。
- 事前分布感度分析: $\epsilon$の事前分散を4倍に緩めても事後の$R_0$中心はほぼ不変(1.050→1.054)。10年の平衡期データだけでは$R_0$を大きく動かす情報がデータ自体に無いことを確認。

### SIRS (米国季節性インフルエンザ ILI、2018-2026)

- $\beta(t)=\beta_0(1+\delta\cos(2\pi t/52-\phi))$で季節性を導入。`pytensor.scan`の`sequences`引数で時間変動パラメータを実装。
- $\beta_0,\delta$の組み合わせで$R_0(t)$の変動幅が過大になり、短周期の異常スパイクが発生。数値積分精度の問題という仮説を反証した上で、$R_0=0.8+\epsilon$+$\delta$の分散調整で収束。
- 最終的に$\gamma\approx0.346$(症状残存期間約3週間)、$\xi\approx0.0375$(免疫持続半年強)など、当初の医学的想定から大きくデータに更新された値を得た。

### SEIR (湖北省COVID-19初期流行、2020)

- $E$(潜伏期)が観測データに直接現れない部分観測系。観測モデルは$\text{daily\_new}\approx\sigma E(t)$。
- $R_0$を1.1→3.5、世代時間を非現実的な水準まで短縮、$E_0$を10倍以上に引き上げても、実データの急峻な立ち上がり(1月末〜2月頭でピーク)を再現できず。
- **結論**: パラメータ調整では解決しない、モデルの生成過程(生物学的拡大)とデータの生成過程(検査・報告体制の拡大)の不整合(モデルの誤設定)である可能性が高いと判断。非識別性とは異なる、より根源的な限界。

## 横断的な学び

- **診断の三段構え**: prior predictive check → サンプリング診断(r_hat, ess, divergence) → 表現力の検証(posterior predictive)、を毎回徹底。
- **Divergence=0でもr_hatが高い場合はチェーン間の平均を比較し、マルチモダリティかチェーン長不足かを切り分ける。**
- **「分母のパラメータが0に近づくと分散が発散する」病理は分布族を問わず繰り返し出現**(SVのσ_η²/(1-φ²)、Gamma-Poissonのμ²/α_conc、Hawkesのβ→0)。0近辺の密度を避ける分布(shape>1のGamma等)への変更が共通の処方箋。
- **2パラメータの"比"が意味を持つ量(R0、分岐比M等)は、個々のパラメータの分散調整では解決しない**。解析的な定常状態の式に立ち返り、直接その比に事前分布を与える再パラメータ化が有効。
- **表面的な改善(有意なパラメータ、収束の良さ)と本質的な問題解決は別軸**。leverage効果の$\rho$が有意でもACFギャップを解消しなかった例、事前分布をいじって見た目を整えることの危険性など、繰り返し確認。
- **非識別性とモデルの誤設定は異なる種類の壁**。前者は再パラメータ化で緩和できるが、後者(SEIRで直面)はパラメータ探索では原理的に解決しない。
- **単位・スケールの見落としはパラメータ推定を丸ごと狂わせる**(サメのGamma分布パラメータ制約、SISの人口正規化、SIRSの週/年単位)。数式に立ち返った検算が繰り返し誤りを発見する鍵になった。
