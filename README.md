# bayesian-epidemiological-models

PyMCによるベイズ機構論的感染症モデリング。SIR→SIS→SIRS→SEIRの順に、コンパートメントモデルの複雑さを段階的に引き上げながら実装した4本のプロジェクトをまとめる。各モデルとも「数式設計 → 事前分布のスケール決定 → prior predictive check → サンプリング → 診断 → 表現力の検証」という共通の手順で取り組んだ。

## SIR: Eyamペスト流行(1666)

**データ**: Raggett(1982)による古典的データセット。8時点(0〜4ヶ月、0.5ヶ月間隔)、村の総人口261人。

$$\frac{dS}{dt}=-\beta SI/N,\qquad \frac{dI}{dt}=\beta SI/N-\gamma I$$

- `pytensor.scan`によるオイラー法数値積分を実装。観測間隔(0.5ヶ月)を50ステップに細分化。
- 観測尤度はPoisson分布(S, Iとも「村の記録という同一観測プロセス」という理由で分布族を統一)。
- $\beta$が$SI/N$という積の形で効くことを踏まえ、データから$\beta\approx5.5$と検算。Raggett原論文の推定値($\hat\beta=0.0212$、$N$で割らない定義)を$N\approx261$倍した値とほぼ一致し、独立した整合性チェックとなった。

**結果**: $\beta\approx5.04$, $\gamma\approx3.17$, $R_0=\beta/\gamma\approx1.59$。Raggett(1982)の推定値($R_0\approx1.9$)と近いオーダーで独立に再現。ピーク時点(t=1.5ヶ月)は完全一致、ピーク高さ(実測29 vs 予測中央値24.7)はやや過小評価。Poisson観測分布の「分散=平均」という制約が、急峻なピーク形成を説明しきれなかった可能性。

## SIS: ナイジェリア マラリア罹患率

**データ**: World Bank指標(SH.MLR.INCD.P3)、ナイジェリア2015-2024年(地方病的平衡に落ち着いた期間)、10点。

$$\frac{dI}{dt}=\beta\frac{(N-I)I}{N}-\gamma I,\qquad \text{observed incidence}\approx\gamma I(t)$$

- 「incidenceはI(t)そのものか、βSI/Nか、γIか」という観測モデルの解釈を検討し、平衡期データという前提から$\gamma I$近似を採用。
- $I_0$を独立にサンプルせず`I0 = incidence_obs[0]/gamma`として`gamma`から導出(値の矛盾を構造的に排除)。
- **単位の見落とし**: 指標が「リスク人口1,000人あたり」であることを見落とし$N=1000$をそのまま使用、$\gamma$の生物学的妥当性とデータの平衡水準が両立しない事態に。実際のリスク人口(2×10^8)へ変換して解決。
- 平衡水準の解析式$\text{incidence}^*=\gamma N(1-1/R_0)$に実データを代入し、必要な$R_0\approx1.05$を逆算。$R_0=1+\epsilon$、$\epsilon\sim\text{Gamma}(\text{平均}0.05)$という再パラメータ化で暴走(prior predictiveの最大値が現実の数百万倍)を解消。

**結果**: $R_0\approx1.050$〜$1.054$(事前分散を4倍に緩めても中心はほぼ不変)。10年の平衡期データのみでは、$R_0$を大きく動かすだけの情報がデータ自体に構造的に存在しないことを事前分布感度分析で確認。

## SIRS: 米国季節性インフルエンザ

**データ**: CDC FluView(ILINet)、週次`% WEIGHTED ILI`、2018-2026年、444週。

$$\beta(t)=\beta_0(1+\delta\cos(2\pi t/52-\phi))$$
$$\frac{dS}{dt}=-\beta(t)SI/N+\xi R,\quad \frac{dI}{dt}=\beta(t)SI/N-\gamma I,\quad \frac{dR}{dt}=\gamma I-\xi R$$

- `pytensor.scan`の`sequences`引数(ステップごとに異なる値を渡す)を用いて季節変動する$\beta(t)$を実装。$N=100$(パーセント表記に対応)。
- **病理の診断プロセス**: prior predictiveで短周期の異常スパイクが発生。「時刻配列の単位誤り」「オイラー法の数値不安定性」という2つの仮説を、コード確認と`n_steps_per_week`を10→50に増やす実験でそれぞれ反証。最終的に$\beta_0,\delta$の組み合わせで$R_0(t)_{\max}\approx4.5$まで暴走していたことを検算で特定。
- $R_0=0.8+\epsilon$への再パラメータ化と$\delta$の事前分散の大幅な引き締め(Beta(30,70)、平均0.3)により、$R_0(t)_{\min}<1$(季節性の"谷")を確保して収束。

**結果**: $\gamma\approx0.346$(症状残存期間約2.9週、事前想定の1週間から大きく上方修正)、$\xi\approx0.0375$(免疫持続約半年強)、$R_0\approx1.362$、$\delta\approx0.189$、$\phi\approx-1.146$rad(ピーク週は10月末〜11月頭、実際の流行開始時期と整合)。ただし$R_0(t)_{\min}\approx1.10$で1を上回ったままとなり、「観測データは夏に明確に減少するがR0(t)は年間を通じ1超」という一見矛盾する結果を、"R0(t)の符号"と"観測ILIの絶対水準"は別の情報である、という考察を添えて次回への課題として保留。

## SEIR: 湖北省COVID-19初期流行

**データ**: JHU CSSE COVID-19データリポジトリ、湖北省日次新規感染者数、2020年1月22日〜3月31日、70日分。

$$\frac{dE}{dt}=\beta SI/N-\sigma E,\qquad \frac{dI}{dt}=\sigma E-\gamma I,\qquad \text{daily\_new}\approx\sigma E(t)$$

- $E$(潜伏期)が観測データに直接現れない部分観測系という、これまでで最も難易度の高い構造。
- **段階的な反証プロセス**: ①$R_0$を1.1→2.5→3.5と引き上げても立ち上がりのタイミングは改善せず、②$\sigma,\gamma$を潜伏・感染期間2日という非現実的な水準まで縮めても実データの急峻さに届かず、③$E_0$を独立確率変数化(LogNormal、機械的逆算値の10倍以上)しても変化はごくわずか。

**結果**: 主要な自由度(R0、世代時間、E0)を全て試し尽くしても実データの急拡大(1月末〜2月頭でピーク)を再現できず。**「観測データが真の生物学的感染拡大速度ではなく、検査・報告体制の急速な拡充速度を反映している」**という、モデルの生成過程とデータの生成過程そのものの不整合(モデルの誤設定)が根本原因である可能性が高いと判断。パラメータ探索では原理的に解決しない種類の限界として記録。

## 横断的な学び

1. **事前分布のスケールはデータから逆算する**: EDAで観測される増減速度・平衡水準・季節振幅を、モデルの解析的な定常状態の式に代入して事前分布の中心値を決める、という手順が4モデル共通で有効だった。
2. **「γ = 期間」ではなく「1/γ = 期間」という取り違えが繰り返し発生**: SIS・SEIRの双方で同型の誤りが起き、都度「単位」に立ち返ることで修正した。
3. **2パラメータの比(R0)が意味を持つ量は、個々のパラメータの分散調整では解決しない**: SIS・SIRSともに$R_0=(\text{定数})+\epsilon$という再パラメータ化が、prior predictiveの暴走を解消する決め手になった。
4. **単位・正規化の見落としはパラメータ推定を丸ごと狂わせる**: SISの「1,000人あたり」指標の見落としが典型例。実際の人口スケールへの変換で解決。
5. **仮説を反証しながら原因を絞り込む診断プロセス**: SIRS・SEIRの双方で、複数の候補仮説(単位誤り、数値積分精度、パラメータの組み合わせ、観測開始時点の遅れ)を一つずつ実験で確かめ、消去法で真因に迫った。
6. **非識別性とモデルの誤設定は異なる種類の壁**: 前者(SIS・SIRSのR0まわり)は再パラメータ化で緩和できたが、後者(SEIR)はモデルが想定する生成過程とデータの実際の生成過程が食い違っており、パラメータ探索では原理的に解決しない。「モデルが健全に収束すること」と「モデルが正しい問いに答えていること」は別軸であるという原則の、最も強い形での確認となった。
